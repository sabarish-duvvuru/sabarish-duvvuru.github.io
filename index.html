Evidence evolution

The work starts as scattered engineering evidence. The value is converting it into records, maps, and review packets that humans can verify.

MD/MMD operating model: Markdown captures the evidence. Mermaid maps dependencies and handoffs. Human review controls the decision. The result is a lightweight engineering record that can be inspected, revised, and reused.
